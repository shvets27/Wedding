# Как удалить все сообщения бота из Telegram

## ⚠️ Важно понимать

Telegram Bot API позволяет удалять только **сообщения, которые отправил сам бот**. Для удаления нужен `message_id` каждого сообщения.

## Метод 1: Простой способ (рекомендуется)

**Если это личный чат с вами:**

1. Откройте Telegram чат, куда приходят анкеты
2. Нажмите **три точки (⋮)** вверху
3. Выберите **"Очистить историю"**
4. Выберите **"Удалить для всех"** (если доступно)
5. Подтвердите

**Если это группа:**
- Нужны права администратора
- В меню группы → **"Очистить историю"**

## Метод 2: Через Bot API (программно)

### Вариант А: Удаление последних сообщений

Используйте файл `delete-telegram-messages.js`:

1. Откройте файл `delete-telegram-messages.js`
2. Замените `ВАШ_ТОКЕН_БОТА` на ваш токен
3. Запустите: `node delete-telegram-messages.js`

⚠️ **Ограничение:** `getUpdates` возвращает только последние 100 сообщений

### Вариант Б: Удаление по ID (если знаете ID)

1. Получите список `message_id` всех сообщений
2. Используйте Telegram Bot API:

```javascript
const TELEGRAM_BOT_TOKEN = 'ваш_токен'
const TELEGRAM_CHAT_ID = '647597624'
const messageIds = [123, 124, 125, ...] // ID сообщений

for (const messageId of messageIds) {
  await fetch(`https://api.telegram.org/bot${TELEGRAM_BOT_TOKEN}/deleteMessage`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
    body: new URLSearchParams({
      chat_id: TELEGRAM_CHAT_ID,
      message_id: messageId.toString()
    }).toString()
  })
}
```

### Как получить message_id?

**Способ 1:** Через ссылку на сообщение
- Долгое нажатие на сообщение → **"Копировать ссылку"**
- В ссылке будет `message_id`, например: `t.me/username/123`

**Способ 2:** Использовать бота @getidsbot
- Добавьте бота в чат
- Он покажет ID всех сообщений

**Способ 3:** Через getUpdates API
- Только последние 100 сообщений
- Используйте скрипт выше

## Метод 3: Через Telegram Desktop (для личного чата)

1. Откройте Telegram Desktop
2. Найдите чат
3. Правой кнопкой на чат → **"Удалить чат"**
4. Или выделите все сообщения и удалите

## Рекомендация

Для свадебного сайта проще всего:
- Если это **личный чат** → используйте "Очистить историю" в Telegram
- Если это **группа** → используйте админ-меню для очистки

Программное удаление через Bot API полезно только если нужно удалить много старых сообщений автоматически.
